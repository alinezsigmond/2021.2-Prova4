# Como utilizar

É necessário inicializar o backend e o frontend na IDE de sua preferência ou utilizando linha de comando.
<br>O frontend exibirá as seguintes opções: Fornecedores, Tipos de produtos e Produtos. Cada página listará os valores cadastrados em suas respectivas categorias e 
terá a opção de criar, editar e deletar os itens. 
<br>Para controlar o estoque, informe o valor que será incrementado ou decrementado e selecione a opção correspondente.
